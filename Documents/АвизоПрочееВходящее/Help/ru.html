Документ "Авизо по прочим операциям входящее"

Документ предназначен для отражения поступления данных, для которых не предусмотрено других документов авизо (Авизо по расчетам входящее, Авизо по ОС входящее, Авизо по МПЗ входящее).

Поступление данных регистрируется от обособленного подразделения, выделенного на отдельный баланс.

Документ может быть введен на основании документа Авизо по прочим операциям исходящее.

Одним документом можно зарегистрировать данные по нескольким счетам учета с разной аналитикой.

При вводе документа в шапке нужно указать следующие реквизиты:

  • Организация - обособленное подразделение, которому передаются учетные данные.
  • Отправитель - обособленное подразделение, с баланса которого были переданы учетные данные.
  • Счет расчетов - счет внутрихозяйственных расчетов, субсчет счета 79 "Внутрихозяйственные расчеты".

По принимаемым на баланс суммам нужно указать Счет учета и аналитику по этому счету.

Принятые суммы можно заполнить автоматически по документу-основанию по кнопке Заполнить по документу-основанию.
