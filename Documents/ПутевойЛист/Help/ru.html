Документ "Путевой лист"

Документ предназначен для отражения расхода топлива по путевым листам для нетранспортных компаний, которые используют автомобиль в служебных целях.


Заполнение документа

Топливо

  • Документ - вид документа, по которому была осуществлена покупка топлива: по кассовому чеку или по топливной карте.
  • Номер - номер кассового чека или номер топливной карты.
  • Дата - дата покупки топлива.
  • Количество - количество купленного топлива.
  • Цена, Сумма НДС, Сумма заполняется из данных кассового чека. Для топливной карты поля не заполняются.

Маршрут

  • Пункт отправления/назначения - указывается адрес пунктов по маршруту следования автомобиля.
  • Дата, Время - дата и время начала/окончания поездки от пункта отправления до пункта назначения.
  • Показания одометра заполняются на основании данных автомобиля по маршруту от пункта отправления до пункта назначения.
  • Расстояние заполняется в километрах по пройденному маршруту от пункта отправления до пункта назначения.
  • Расход - расход топлива, затраченный по пройденному маршруту от пункта отправления до пункта назначения.

Печатные формы

Для документа Путевой лист предусмотрены следующие печатные формы:

  • Унифицированная форма путевого листа №3
  • Упрощенная форма путевого листа легкового автомобиля
  • Авансовый отчет АО-1
  • Реестр документов путевых листов
