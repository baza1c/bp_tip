Документ "Списание товаров, материалов"

Документ предназначен для отражения списания оборудования, товаров, материалов и готовой продукции на недостачи, потери от порчи ценностей и списания выданных сотрудникам материалов. Документ можно ввести на основании документа Инвентаризация товаров на складе.

Печатные формы

Для операции Списание со склада при вводе документа в шапке обязательно нужно указать Склад, с которого осуществляется списание. Для операции Списание с сотрудника обязательным является реквизит Сотрудник.

Списание товаров, материалов, готовой продукции и оборудования

Списания товаров, материалов, готовой продукции и оборудования отражается операцией Списание со склада. Для списания необходимо заполнить закладку Товары.

На закладке Товары указывается номенклатура, количество и счет учета.

  • Поле Счет учета заполняются автоматически при указании номенклатуры. Информация для заполнения берется из регистра Счета учета номенклатуры.
  • При списании товаров из неавтоматизированной розничной торговой точки, в которой учет товаров ведется по продажным ценам, необходимо дополнительно заполнить поле Розн. цена (руб.).

Списание тары

Отражается операцией Списание со склада. Для отражения списания тары необходимо заполнить закладку Тара.

На закладке Тара указывается номенклатура, количество и счет учета.

  • Поле Счет учета заполняются автоматически при указании номенклатуры. Информация для заполнения берется из регистра Счета учета номенклатуры.

Списание материалов с сотрудника

Списание выданных сотрудникам материалов отражается операцией Списание с сотрудника. 

В шапке документа нужно указать Сотрудника, с которого списываются материалы. В таблице указываются Номенклатура для списания и Количество. 

Печатные формы

Для документа Списание товаров предусмотрены следующие печатные формы:

  • Акт списания товаров
  • ТОРГ-16 (Акт о списании товаров)
