Документ "Авизо по МПЗ исходящее"

Документ предназначен для передачи товарно-материальных ценностей между обособленными подразделениями, выделенными на отдельный баланс.

При вводе документа в шапке нужно указать следующие реквизиты:

  • Организация - обособленное подразделение, на котором числится передаваемая задолженность.
  • Получатель - обособленное подразделение, на баланс которого передается задолженность.
  • Подразделение - производственное подразделение, на котором числятся МПЗ.
  • Счет расчетов - счет внутрихозяйственных расчетов, субсчет счета 79 "Внутрихозяйственные расчеты".

При передаче нужно заполнить закладки:

  • Товары
  • Авизо

На закладке Товары нужно заполнить список товарно-материальных ценностей.

  • Заполнить список можно по кнопке Подбор.
  • Предоставляется возможность дополнить список номенклатурой, например, из документа поступления, установить номер ГТД или страну происхождения. Эти действия можно выполнить по кнопке Изменить.

На закладке Авизо заполняются данные бухгалтерского и налогового учета товаров, указанных на закладке Товары (себестоимость, партия и т.д.).

  • Закладка заполняется автоматически по кнопке Заполнить авизо закладки Товары. После заполнения данные при необходимости можно скорректировать.

Для документа Авизо по МПЗ исходящее предусмотрена печатная форма:

  • Авизо

На основании Авизо по МПЗ исходящее можно ввести документ:

  • Авизо по расчетам входящее
