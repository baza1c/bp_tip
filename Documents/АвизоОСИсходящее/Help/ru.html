Документ "Авизо по ОС исходящее"

Документ предназначен для передачи основных средств между обособленными подразделениями, выделенными на отдельный баланс.

При вводе документа в шапке нужно указать следующие реквизиты:

  • Организация - обособленное подразделение, на котором числится основное средство.
  • Получатель - обособленное подразделение, на баланс которого передается основное средство.
  • Счет расчетов - счет внутрихозяйственных расчетов, субсчет счета 79 "Внутрихозяйственные расчеты".
  • Событие - событие основного средства, которое отражается документом.

В табличной части необходимо заполнить список переданных основных средств и сведения о них.

  • Заполнить список можно по кнопке Подбор.
  • Сведения об основном средстве можно заполнить по кнопке Заполнить данные ОС.

Для документа Авизо по ОС исходящее предусмотрена печатная форма:

  • Авизо

На основании Авизо по ОС исходящее можно ввести документ:

  • Авизо по ОС входящее
