Документ "Авизо по расчетам исходящее"

Документ предназначен для передачи дебиторской и кредиторской задолженностей между обособленными подразделениями, выделенными на отдельный баланс.

Одним документом может быть передано несколько сумм дебиторской и кредиторской задолженности по разным контрагентам, договорам и документам расчетов.

При вводе документа в шапке нужно указать следующие реквизиты:

  • Организация - обособленное подразделение, на котором числится передаваемая задолженность.
  • Получатель - обособленное подразделение, на баланс которого передается задолженность.
  • Счет расчетов - счет внутрихозяйственных расчетов, субсчет счета 79 "Внутрихозяйственные расчеты".

В табличной части нужно заполнить реквизиты:

  • Вид задолженности - выбрать значение из списка: Дебиторская или Кредиторская.
  • Контрагент - контрагент, задолженность по которому передается.
  • Договор - договор, задолженность по которому передается.
  • Документ расчетов - документ расчетов, задолженность по которому передается.
  • Счет учета - счет учета, на котором числится передаваемая задолженность.
  • Сумма - сумма передаваемой задолженности.

Для документа Авизо по расчетам исходящее предусмотрена печатная форма:

  • Авизо
