Настройки сервиса интеграции

Форма позволяет установить настройки сервиса интеграции:

Наименование сервиса
В поле отображается имя сервиса интеграции, как оно задано в конфигурации.
Имя пользователя 
В параметре можно выбрать имя пользователя, от имени которого будет выполняться обработка полученных сообщений. Пользователь должен иметь право "Использование" для канала сервиса интеграции. Если пользователь не указан, то обработка полученных сообщений происходит с правами, которые определяются набором ролей, указанных в свойстве конфигурации "Основные роли". В том случае, если в этом свойстве не указано ни одной роли, выполнение происходит без ограничения прав доступа.

Настройки внешнего сервиса интеграции

Адрес 
URL-адрес, по которому доступен сервис интеграции.
Имя пользователя 
Имя пользователя, под которым выполняется соединение с сервисом интеграции.
Пароль пользователя 
Пароль пользователя, под которым выполняется соединение с сервисом интеграции.

При нажатии кнопки OK настройки будут применены и форма настроек будет закрыта.
